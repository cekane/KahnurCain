# Personal Website

## Getting started
After the repository is cloned, install all the dependicies with: 
```sh
npm install
```
Now start the node server with:
```sh
npm start
```

## Tech
1. React - home page is desgined using react components
2. SCSS - Styles
3. npm - package manager
